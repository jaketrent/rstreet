# Rstreet

Smart Uploader for AWS S3 buckets

Ruby port of [tgroshon's](https://github.com/tgroshon) [street.js](https://github.com/tgroshon/street.js)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rstreet'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rstreet
